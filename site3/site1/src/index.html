React.js

React.js는 사용자 인터페이스(UI)를 구축하기 위해 사용되는 JavaScript 라이브러리입니다. Facebook에서 개발한 React는 단일 페이지 애플리케이션(SPA) 및 대규모 애플리케이션의 UI 개발을 간편하게 만들기 위해 설계되었습니다.

Node.js

Node.js는 이벤트 기반, 비동기식 프로그래밍 방식을 채택하여 빠른 네트워크 애플리케이션 개발과 확장성 있는 서버를 구축할 수 있도록 지원합니다.

Vite.js

Vite.js는 Vue.js 프레임워크를 위한 개발 도구로서, 빠른 개발 서버와 최신 웹 개발 기술을 지원하는 빌드 도구를 제공합니다. Vite.js는 새로운 웹 개발 경험을 제공하기 위해 개발되었습니다..
