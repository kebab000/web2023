  • 브라우저 속성 : window.innerWidth : 브라우저 가로값(스크롤바 미포함)
  • 브라우저 속성 : window.innerHeight : 브라우저 세로값(스크롤바 미포함)
  • 브라우저 속성 : window.outerWidth : 브라우저 가로값(스크롤바 포함)
  • 브라우저 속성 : window.outerHeight : 브라우저 세로값(스크롤바 포함)
  • 브라우저 속성 : window.screen.width : 모니터의 가로값
  • 브라우저 속성 : window.screen.height : 모니터의 세로값
  • 브라우저 속성 : window.pageXOffset : 가로 스크롤 값(브라우저 기준)
  • 브라우저 속성 : window.pageYOffset : 세로 스크롤 값(브라우저 기준)
  • 브라우저 속성 : window.scrollX : 가로 스크롤 값(브라우저 기준)
  • 브라우저 속성 : window.scrollY : 세로 스크롤 값(브라우저 기준)
  • 문서 속성 : document.documentElement.scrollY : 세로 스크롤 값(브라우저 기준)
  • 문서 속성 : document.documentElement.scrollX : 가로 스크롤 값(브라우저 기준)
  • 문서 메서드 : document.body.offsetHeight() : 문서의 전체 높이
  • 브라우저 메서드 : window.scroll() : 스크롤 값 설정
  • 브라우저 메서드 : window.scrollTo() : 스크롤 값 설정(절대 기준)
  • 브라우저 메서드 : window.scrollBy() : 스크롤 값 설정(상대 기준)
  • 요소 속성 : element.scrollLeft : 가로 스크롤 값
  • 요소 속성 : element.scrollTop : 가로 스크롤 값
  • 요소 메서드 : element.scrollIntoView() : 스크롤 이동 및 설정
  • Tip! : (window.innerHeight + window.scrollY) >= document.body.offsetHeight() : 현재 스크롤 위치(window.scrollY)와 브라우저 창의 높이(window.innerHeight)를 합친 값이 문서의 전체 높이(document.body.offsetHeight)와 같거나 큰 경우에만 아래 코드를 실행한다. 즉, 사용자가 페이지의 맨 아래에 도달했을 때를 의미.
